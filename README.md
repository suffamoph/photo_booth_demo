# 咔咔AI照相机优化项目

## 项目背景
咔咔AI照相机优化项目旨在为咔咔AI照相机设备提供更智能的拍照体验。通过结合自然语言处理和生成式人工智能（AIGC）技术，用户可以通过语音或文字输入指令，设备将识别用户意图并调用相应的AI功能，生成高质量的照片或合成图像。

### 核心功能
- **自然语言输入**：支持语音和文字输入，方便用户操作。
- **意图识别**：通过自然语言处理技术，理解用户需求。
- **AIGC能力调用**：
  - AI证件照生成
  - AI写真生成
  - IP形象合影合成
  - 虚拟景点打卡
  - 云打印
- **结果返回**：快速生成并返回高质量的图像。

### 技术储备
- **现有技术**：
  - IP形象库
  - 写真模板库
  - AI证件照生成技术
- **待开发模块**：
  - LLM核心处理
  - 意图识别
  - 任务调度

## 项目目标
通过本项目，我们希望实现：
1. 提升咔咔AI照相机的智能化水平。
2. 为用户提供更便捷、更高效的拍照体验。
3. 探索LLM和意图识别技术在实际场景中的应用。

## 项目规划

### Demo开发目标
本项目的目标是开发一个小型Demo，能够在PC本地跑通核心流程，验证技术可行性。Demo将以证件照生成为例，模拟用户拍照并生成结果。

### 技术栈
- **硬件环境**：
  - GPU：RTX 4090
  - RAM：64G
  - CPU：i7 13700F
- **软件环境**：
  - Python
  - Node.js
  - Ollama（调用本地LLM）
  - OpenAI API（调用云端LLM）
  - ComfyUI（本地部署，用于生成图像）

### Demo功能模块

#### 后台能力
1. **意图理解**
   - 支持以下意图：
     - 纯聊天
     - 证件照生成
     - 写真照生成
     - IP合影
     - 虚拟打卡
     - 云打印
2. **任务调度**
   - 根据用户意图调用相应的模块。
3. **素材管理**
   - 管理IP形象库、写真模板库、虚拟背景素材。
4. **模型和接口管理**
   - 管理本地和云端模型的调用接口。

#### 前端能力
1. **自然语言输入**
   - 提供文本域，支持用户输入自然语言指令。
2. **照片上传与结果展示**
   - 用户可上传照片，后台处理后返回生成结果。
3. **实时进度展示**
   - 通过对话框实时展示任务处理进度。

### Demo开发步骤
1. **需求分析**
   - 明确Demo的功能范围和技术选型。
2. **环境搭建**
   - 配置Python和Node.js开发环境。
   - 安装必要的依赖库。
3. **模块开发**
   - **前端开发**：实现自然语言输入、照片上传和进度展示功能。
   - **后台开发**：
     - 实现意图理解模块。
     - 集成ComfyUI，完成证件照生成功能。
     - 实现任务调度逻辑。
4. **测试与优化**
   - 测试Demo的功能完整性和稳定性。
   - 根据测试结果优化代码。
